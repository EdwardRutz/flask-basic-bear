# Basic Bear Generator

>Python, Flask, a simple app to build a basic bear


## Depedencies

- Python
- Flask



## Setup

- Install Flask, `pip install Flask`
- Check Flask version in REPL
```python
python      # Starts REPL
import flask
flask.__version__
```
